Knights Adventure

Download

Para download, use os links que se encontram no cabeçalho. Ambas versões (Para Linux e para Windows) estão disponibilizadas no mesmo arquivo.

Instalação

Ambas as versões do jogo se encontram no arquivo. Mas elas são instaladas de forma diferente, de acordo com o seu sistema operacional. Se você está usando

Linux

Tenha certeza de que você possui o pacote lua instalado no seu computador, caso não possua, digite, no terminal (como root):

guest@computer:~$ apt-get install lua

O terminal vai lhe dar várias versões diferentes para instalar, por exemplo:

lua5.1 Lua 5.1
lua5.2 Lua5.2.1
lua5.0.1 Ubuntu-generic1

Escolha SEMPRE a versão mais nova (nesse caso, a versão 5.2), então digite:

apt-get install lua5.2

Siga o que o programa diz, então, depois do pacote lua instalado, execute o programa unixinstall.sh, o programa vai instalar e configurar o Knight Adventures. Depois disso, execute o programa unixlauncher.sh Se tudo correr bem , depois disso você poderá executar o jogo tranquilamente. Entretanto, caso você esteja usando

Windows

Será necessário que você tenha o LuaForWindows instalado no computador. Assim que instalado, extraia e execute o programa makeInstall.bat. Depois que o programa for instalado, execute o programa w32launcher.bat. Se tudo estiver correto, o jogo deverá funcionar corretamente.
